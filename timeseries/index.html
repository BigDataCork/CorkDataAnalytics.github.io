As discussed above there is a seasonal pattern to the planning permissions data. If there was a relationship, one might anticipate a similar seasonal pattern with the live register data. However although there is initial evidence of a similar pattern, the number leaving the live register is not as closely linked to the number of planning permissions granted as time goes on. This could have been because for the initial quarters recruitment started as soon as permission was granted. It could also have been coincidental as workers were signing off the register for different reasons such as emigration or work elsewhere in the country and the patterns just happened to match at this time. Later planning permissions granted did not seem to have the same impact on the live register. Indeed although planning permissions retained their seasonality, the number of construction workers signing off the live register continued on a decline.

However, let’s back this visual analysis with some regression analysis.

assumptions:

  1. independent and identically distributed (iid) normal variables.

  2. observations between quarters are independent.

  3. the time series covered by the data is indicative of a broader time scale.

Generally speaking there is a doubt about the autocorrelation of errors in time series data. In this case however, the numbers leaving the live register and number of permissions granted can be assumed to be independent of previous quarters.

Planning permission approvals for Multi development houses can be used to predict the numbers leaving the live register data for Cork city. However, because of the low R-squared value we should do some further analysis on our data.

# Evaluate Collinearity
durbinWatsonTest(lm2)
##  lag Autocorrelation D-W Statistic p-value
##    1       0.5928672     0.4884617       0
##  Alternative hypothesis: rho != 0

The p-value in the durbin Watson test is zero, so we can conclude in fact that there is auto-correlation amongst the residuals between live register and multi house developments. It was assumed above that the number signing off the live register was indepdendent of the previous quarter, in that unlike currencies for example, the number signing off the live register was not related to the previous quarter. This is because they were not following on from the previous quarter, but rather were new. However, our evidence suggests there is a high correlation between the errors for the live register regressed on multi house developments. As both of these sets of measurements are per quarter, one can therefore infer that the cyclical nature of permissions granted does impact on the numbers signing off the live register. In other words there is a dependence between the number of permissions granted and the number leaving the live register per quarter. Steps to remedy this auto-correlation would be to:

extend the time periods from quarterly to half-yearly or yearly

include a linear term to ‘smooth’ out the cyclicalities

include seasonal dummy variables, where at any point in the time series, t, one variable will equal 1 and the other variables 0

Use transformed Variables that remove the (estimated) autocorrelation parameter (Cochrane-Orcutt and Hildreth-Lu Procedures)

Use First Differences, where the first difference of Y at period t is equal to Yt-Yt-1

Use a more sophisticated forecasting model such as exponential smoothing or ARIMA may be appropriate

Use Estimated Generalized Least Squares

So it looks like we can’t trust the results from the analysis above: the regression model is likely to be seriously biased or misleading.

But, although we know that we can’t use the model, let’s see what emerges from the diagnostic tests below.

The Shapiro-Wilk normality test shows that the p-value is greater than 5%, so we can conclude that the residuals follow a normal distribution and thus sum to zero. This is corroborated by the histogram which suggests a normal distribution.

The small number of observations is noticeable in these plots.

The scatter smooth suggests a non-linear negative relationship that is greatly influenced into becoming postive by the outlier. The variance appears to be constant but systematic- so the errors are not independent.

In the plot of residuals vs fitted values the residuals do not bounce randomly around the 0 line. Infact there is evidence that certain values appear to be grouped around certain points in time, i.e. cyclicality. This suggests that the assumption that the relationship is linear is not strong.

However, the residuals roughly form a “horizontal band” around the 0 line. This suggests that the variances of the error terms are equal,even if they are not independent.

One residual does “stand out” from the basic random pattern of residuals. This suggests that there is an outlier and that it is likey to have high leverage.

In the normal Quantile-Quantile plot, there are slight departures above and below the reference line, however on balance the residuals appear to follow a normal distribution.

We can conclude that the relationship is not linear and the errors are not independent and are not randomly distributed. So we can’t use this model.

What would we do with the outlier if we could use the above model ?

We should not remove the outlier as the data period and the number of obserations are quite small. For example the data period does not cover the Celtic Tiger period where it is quite likely that 14 or more mutli developments were approved quite often, thus also having an effect on the live register- however, we need also to remember that at that period there was also recruitment from other countries, so it also possible that the effect on the register was minimal. In essence rather than remove the outlier from the data, it is safer to conclude that the data set is too small. While there are indications that the higher the number of multi development permissions granted for Cork city in a quarter- the higher the number of construction workers in Cork city that sign off the live register- the above diagnostics would query the use of a data set of this size (and perhaps time intervals) for such predictions.

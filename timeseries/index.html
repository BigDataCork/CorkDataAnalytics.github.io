As discussed above there is a seasonal pattern to the planning permissions data. If there was a relationship, one might anticipate a similar seasonal pattern with the live register data. However although there is initial evidence of a similar pattern, the number leaving the live register is not as closely linked to the number of planning permissions granted as time goes on. This could have been because for the initial quarters recruitment started as soon as permission was granted. It could also have been coincidental as workers were signing off the register for different reasons such as emigration or work elsewhere in the country and the patterns just happened to match at this time. Later planning permissions granted did not seem to have the same impact on the live register. Indeed although planning permissions retained their seasonality, the number of construction workers signing off the live register continued on a decline.

However, let’s back this visual analysis with some regression analysis.

assumptions:

  1. independent and identically distributed (iid) normal variables.

  2. observations between quarters are independent.

  3. the time series covered by the data is indicative of a broader time scale.

Generally speaking there is a doubt about the autocorrelation of errors in time series data. In this case however, the numbers leaving the live register and number of permissions granted can be assumed to be independent of previous quarters.

lm<-lm(newdata$livereg~newdata$planning)

Multiple R-squared: 0.1245

12.45 % of the variability in the live register can be explained by the regression on the planning permissions data. This is low and suggests a poor relationship.

F-statistic is 1.99/t- value is 1.411 .

P value ( 0.1801) is greater than 5% so accept null hypothesis. There is no relationship between the planning permissions and the numbers leaving the live register.

Planning permissions granted data should not be used to predict the numbers leaving the live register data for Cork city.

What can we take from this ?

Just because planning permission has been given in a certain quarter doesn’t mean that the work will start in that quarter or at all.

Construction workers signing off the live register in Cork are doing so for construction jobs in more than just Cork city.

The planning data is confined to residential property, so perhaps if commercial planning permissions were included, it may change the output.
